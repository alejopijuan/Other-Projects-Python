    parameter.

    The function should always return None.

    If the file is run, it should run the code provided in the intial LabD3.py
    file, and out exactly:

sort_list(['the', 'cat', 'in', 'the', 'hat', 'is', 'back'])

Actual answer: ['back', 'cat', 'hat', 'in', 'is', 'the', 'the']
